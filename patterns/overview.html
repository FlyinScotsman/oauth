üéØ Quick Pattern Selection Guide

Need User Login?

Choose A-series patterns

  • A1: Fast web apps
  • A2: Enterprise control
  • A3: SPA/Mobile
  • A4: Devices/IoT

Service-to-Service?

Choose B-series patterns

  • B1: High performance
  • B2: Enterprise governance
  • B3: Simple development

JWT vs Opaque?

Key trade-offs

  • JWT: Performance
  • Opaque: Control
  • Both: Refresh tokens
